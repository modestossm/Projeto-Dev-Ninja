O Mítico Homem-Mês: Ensaios Sobre Engenharia de Software, de Frederick P. Brooks Jr.

R$ 99,99

"O Mítico Homem-Mês", de Frederick P. Brooks Jr., é um clássico da engenharia de software que expõe os desafios reais da gestão de grandes projetos. A obra questiona a crença de que adicionar mais pessoas acelera projetos atrasados, a célebre "Lei de Brooks", mostrando que certos trabalhos não podem ser paralelizados e que comunicação entre equipes aumenta a complexidade.

O livro enfatiza que manter a integridade conceitual de um projeto é fundamental para seu sucesso, e que não existem soluções mágicas para os problemas do desenvolvimento de software. Baseado na experiência de Brooks gerenciando o OS/360 da IBM, o livro permanece uma referência essencial para quem trabalha com gestão de projetos complexos, oferecendo insights práticos sobre comunicação, planejamento e qualidade que transcendem décadas.

O Programador Pragmático: De Aprendiz a Mestre, de Andrew Hunt e David Thomas

R$ 99,99

Este livro é um guia prático que vai além da técnica pura, propondo uma filosofia de desenvolvimento profissional baseada em 70 dicas independentes. O livro aborda responsabilidade pessoal, aprendizado contínuo, testes, refatoração e automação, enfatizando que um bom programador é adaptável, engenhoso e comprometido com qualidade. Com uma abordagem pragmática que equilibra excelência técnica com realidade comercial, apresenta princípios atemporais como o DRY (Don't Repeat Yourself) que permanecem relevantes até hoje, tornando-o leitura essencial para programadores em qualquer nível.

Arquitetura Limpa: O Guia do Artesão para Estrutura e Design de Software, de Robert C. Martin

R$ 99,99

"Arquitetura Limpa", de Robert C. Martin, estabelece princípios universais de design em camadas que protegem a lógica de negócio de detalhes técnicos, criando sistemas escaláveis e testáveis. O livro integra os princípios SOLID com padrões práticos, mostrando como uma boa arquitetura aumenta produtividade ao permitir desenvolvimento paralelo e manutenção sustentável de código limpo. Uma referência indispensável para quem busca criar software resiliente e adaptável

Lógica de Programação e Algoritmos com Javascript

por Edécio Fernando Iepsen

Lógica de Programação e Algoritmos com Javascript - 2ª Edição serve como um guia ideal para quem deseja dominar a lógica de programação por meio de explicações claras e projetos práticos. Seja você um iniciante ou alguém revisitando conceitos, este livro usa exemplos relacionáveis como carros para simplificar ideias complexas.

Você irá mergulhar em JavaScript e Node.js, construindo aplicações tanto front-end quanto back-end. O autor, Edécio, traz uma clareza de nível professor que ajuda você a compreender e aplicar a lógica de programação de forma eficaz.

Refatoração

por Martin Fowler

Por mais de vinte anos, programadores experientes no mundo inteiro contaram com o livro Refatoração de Martin Fowler para aperfeiçoar o design de códigos existentes e melhorar a manutenibilidade do software, assim como para deixar o código existente mais fácil de entender.

Refatoraçã 2ª edição contém um catálogo atualizado das refatorações e inclui exemplos de código JavaScript bem como novos exemplos funcionais que demonstram a refatoração sem classes. Assim como na edição original, este livro explica o que é refatoração, por que você deve refatorar, como reorganizar um código que precise de refatoração e como fazer isso de forma bem-sucedida, independentemente da linguagem usada.

Código Limpo

por Robert C. Martin

Mesmo um código ruim pode funcionar. Mas se ele não for limpo, pode acabar com uma empresa de desenvolvimento. Perdem-se a cada ano horas incontáveis e recursos importantes devido a um código mal escrito. Mas não precisa ser assim.

O renomado especialista em software, Robert C. Martin, apresenta um paradigma revolucionário com Código limpo: Habilidades Práticas do Agile Software. Martin se reuniu com seus colegas do Mentor Object para destilar suas melhores e mais ágeis práticas de limpar códigos “dinamicamente” em um livro que introduzirá gradualmente dentro de você os valores da habilidade de um profissional de softwares e lhe tornar um programador melhor –mas só se você praticar.
